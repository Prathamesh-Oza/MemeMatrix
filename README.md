# MemeMatrix

A robust meme browser where users can search for famous templates and see how they are used.

## Core Features
- **Searchable Meme Grid**: Browse and search through popular meme templates.
- **Category Tabs**: Filter memes by different categories.
- **Share Functionality**: Quick "Share" button to copy the meme link directly to your clipboard.
- **Back to Top**: Intuitive "Back to Top" button logic for easy navigation.
- **Optimized Layout**: Proper handling of diverse image aspect ratios to ensure a clean grid display.

## API Integration
This application is powered by the public MeowMeow API.
